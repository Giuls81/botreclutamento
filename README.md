# Ragnarok Esport Recruitment Bot

Welcome to the digital herald of Ragnarok Esport. This bot is forged to manage recruitment with the professionalism and mythic tone of 'The Valhalla of Sim Racing'.

## Features
- **Slash Command `/apply`**: Triggers a professional Modal for recruitment.
- **Modal Logic**: Collects previous team, motivation, background, and PSN ID.
- **Epic Embeds**: Sends formatted applications to the staff channel in #D4AF37 Gold.

## Quick Start
1. **Prerequisites**: [Node.js](https://nodejs.org/) installed.
2. **Setup**:
   - Rename `.env.example` to `.env`.
   - Fill in your Discord Bot Token and IDs (see [Setup Guide](file:///C:/Users/dalpo/.gemini/antigravity/brain/4bf4f9c3-2f90-482d-a8a8-066cbfdceb41/setup_guide.md)).
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Ignite the Bot**:
   ```bash
   npm start
   ```
